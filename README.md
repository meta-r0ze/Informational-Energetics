This repository contains the latex source for the "Informational Energetics: A Universal Architecture of Persistence" paper.

The paper present a framework to model complex systems that persist using control theory, information theory and thermodynamics. It validates it by deriving the fine-structure constant from first principles and presnting the theoretical foundation for the E8-Persistence Theory.

The rendered version can be viewed here: https://meta-r0ze.github.io/Informational-Energetics/Informational_Energetics__A_Universal_Architecture_of_Persistence.pdf
